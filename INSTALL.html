🚀Quick Install for Windows (Recommended!)

✨ For Windows users - we made it super easy! Our automated PowerShell script does all the work for you.

One-Click Installation

Follow these simple steps:

  1. Download or extract the project folder to your computer
  2. Find the file named install.ps1 in the project folder
  3. Right-click on install.ps1
  4. Select "Run with PowerShell"
  5. Follow the prompts on screen
💡 What the script does:
  • ✓ Checks if you have Node.js installed (installs it if you don't)
  • ✓ Creates all necessary folders
  • ✓ Downloads and installs all required files
  • ✓ Creates a start.ps1 file for easy launching

After Installation

Once the installation is complete, you'll have a start.ps1 file in your folder.

🎉 To use the dashboard:
Just double-click start.ps1 anytime you want to use the dashboard!

Creating a Desktop Shortcut (Recommended)

For even easier access:

  1. Right-click the start.ps1 file
  2. Select "Create shortcut"
  3. Drag the shortcut to your Desktop
  4. Now you can launch the dashboard from your Desktop anytime!
⚠️ If you see a security warning:
Windows might ask if you want to allow the script to run. Click "Yes" or "Allow" - the script is safe and only installs the dashboard.

Mac/Linux Users or Prefer Manual Installation?

No problem! Continue with the manual installation steps below.

🔑Step 3: Configure API Key (Optional)

You can add your OpenAI API key now, or do it later through the dashboard settings.

💡 Recommended for Beginners: Skip this step and add your API key through the dashboard after starting it. It's easier!

Option 1: Using Notepad (Easiest)

  1. Type this command:
notepad .env
  1. When asked "Cannot find the .env file. Do you want to create a new file?", click Yes
  2. Type the following (replace with your actual API key):
OPENAI_API_KEY=sk-your-api-key-here
PORT=3000
  1. Save the file (File → Save)
  2. Close Notepad

Option 2: Using Command Prompt

echo. > .env

Then edit with Notepad as shown above.

Create the configuration file:

touch .env

Open it with a text editor:

nano .env

Add these lines (replace with your actual API key):

OPENAI_API_KEY=sk-your-api-key-here
PORT=3000

Press Control + X, then Y, then Enter to save.

📁Step 4: Create Required Folders

The application needs some folders to store generated images and temporary files.

Copy and paste these commands:

mkdir Images
mkdir data
mkdir uploads
✓ If you see "A subdirectory or file already exists", that's okay! It means the folder is already there.

Copy and paste these commands:

mkdir Images
mkdir -p data
mkdir -p uploads
✓ The -p flag means "create if it doesn't exist, skip if it does."

🔧Troubleshooting Common Issues

Problem: "No API key configured" message

Solution:

  1. Click the ⚙️ Settings button
  2. Enter your OpenAI API key (must start with sk-)
  3. Click Save Settings
  4. Your key is valid if it starts with sk- and is about 50 characters long

Problem: Port 3000 or 5173 is already in use

Solution (Windows):

netstat -ano | findstr :3000
taskkill /PID <PID> /F

Solution (Mac/Linux):

lsof -i :3000
kill -9 <PID>

Problem: "Cannot find module" errors

Solution: Go back to Step 2 and reinstall dependencies.

Problem: Images aren't generating

Check these:

  • Is your API key entered correctly?
  • Does your OpenAI account have credits?
  • Is the server still running? (check the Command Prompt/Terminal)
  • Check the browser console (press F12) for error messages

Windows Only: PowerShell Script Execution Error

Solution:

  1. Open PowerShell as Administrator (right-click → Run as administrator)
  2. Run this command:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  1. Type Y and press Enter
  2. Close PowerShell and try again

Windows Only: Firewall Warning

Solution: When Windows Firewall asks if you want to allow Node.js network access, click "Allow access".

Still having problems? Press F12 in your browser to open Developer Tools and look for red error messages. Take a screenshot and ask for help!
